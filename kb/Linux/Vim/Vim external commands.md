## Comandos Externos Executados Em Modo Normal

| Comando  | Ação                                |
| -------- | ----------------------------------- |
| pwd      | mostra diretorio atual              |
| cd /path | vai para novo diretorio             |
| cd %:p:h | vai para diretorio do arquivo atual |
