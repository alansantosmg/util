## Comandos Marcas

| Comandos | Função                      |
| -------- | --------------------------- |
| ma       | cria marca a em modo normal |
| ’a       | move cursor até marca a     |
| d’a      | delete ate a marca a        |
| y’a      | copia ate a marca a         |
