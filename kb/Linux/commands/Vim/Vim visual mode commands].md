## Comandos Do Modo Visual

| Comandos | Função                                   |
| -------- | ---------------------------------------- |
| v        | seleção de caracteres                    |
| V        | selecão de linhas inteiras               |
| Ctrl V   | seleção de blocos                        |
| d        | deleta selecao                           |
| y        | copia selecao                            |
| c        | altera caracteres                        |
| gu       | minusculas                               |
| shift i  | Insert text (press esc esc after insert) |
