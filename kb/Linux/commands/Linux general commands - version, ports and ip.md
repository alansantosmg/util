# Linux Terminal Util Commands Cheatsheet

| Comando   | Função                               |
| --------- | ------------------------------------ |
| ss -a     | Listar portas abertas no server.     |
| uname -a  | Mostrar versão do kernel do sistema. |
| uname - r | Mostrar versão do kernel sistema.    |
| ip addr   | Listar interfaces de rede.           |
