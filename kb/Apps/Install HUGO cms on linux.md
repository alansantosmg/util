# How to Install Hugo on Linux


### Get Hugo

```bash
mkdir $HOME/src
cd $HOME/src
git clone https://github.com/gohugoio/hugo.git

```

### Instalation

`cd hugoCGO_ENABLED=1 go install --tags extended`
