## Comandos De Manipulação De Abas

| Comando     | Função                   |     |     |
| ----------- | ------------------------ | --- | --- |
| tabnew file | abre arquivo em nova aba |     |     |
| tabnext     | avanca proxima aba       |     |     |
| tabprevios  | retorna aba anterior     |     |     |
