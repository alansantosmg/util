## Comandos Plugin Surround

| Comandos | vim Surround plugin   |
| -------- | --------------------- |
| cs"’     | change “h” to ‘h’     |
| cs’      | change to h           |
| cst      | remove tag inside obj |
| cs}      | { text }              |
| cs{      | {text}                |
| ds"      | remove ""             |
| ysiw]    | [] around the word    |
| yss)     | ([x])                 |
| VS       | textvisual mode       |
| ysiw     | text text             |
| ds{ds)   | remove {} and ()      |
